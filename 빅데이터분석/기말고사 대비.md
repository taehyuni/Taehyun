✅ 2. 결측치 확인 및 제거 : "%in%" 사용
❗ 포인트: 그냥 is.na()가 아니라, %in%을 이용해서 NA를 찾아야 함
```r
# ✅ 정답: is.na()로 확인 후 filter로 제거
library(dplyr)
data %>% filter(!is.na(점수))

# 예: 특정 값이 결측치처럼 취급되는 경우
data <- data.frame(
  이름 = c("철수", "영희", "민수", "지수"),
  점수 = c(90, "없음", 75, "결시")
)

# 점수에서 "없음", "결시" 제거하기 (결측치처럼 취급)
data %>% filter(!(점수 %in% c("없음", "결시")))
```

### 산술연산자
```
1. +, -, *
2. 나누기 : /, %/% , %%
3. / : 나누기 결과를 실수값으로 표현
4. %/% : 몫을 정수로 표현
5. %% : 나머지
```

### 진리값 : TRUE/FALSE
```
& : AND연산(모두가 참인 경우에 참 : 곱하기)
| : OR연산(하나라도 참이면 참 : 더하기)
! : NOT연산(반전)
```
### NA/NULL
```
NA : 잘못된 값이 들어 올 경우(NOT Applicable, Not Available)
NULL : 존재하지 않는 값
```
